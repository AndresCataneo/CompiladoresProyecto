## Requisitos

## Compilación
Desde la raíz del proyecto (`CompiladoresProyecto/ProyectoCompiladores`), ejecutar:
```bash
stack build
```

## Ejecución
Desde la raíz del proyecto (`CompiladoresProyecto/ProyectoCompiladores`), ejecutar:
```bash
stack exec ProyectoCompiladores-exe app/IMP.txt
```
